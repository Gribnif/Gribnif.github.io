Original Shareware Programs

Sat, Jan 20, 2018

These files may be distributed freely. All are in ZIP format, and will work only on Atari and compatible computers and emulators.

WorldClock

A resizable, graphical world clock which tells you the time anywhere in the world. You can even add new locations to the map by simply typing-in the longitude and latitude.

worldclk.zip 2/22/97

MemFile 3.1

MemFile is a hex/decimal editor. It can edit memory, files, and disk sectors. It includes powerful virtual copying and searching.

memfil31.zip 2/22/97

ColaCalc 1.3

ColaCalc is a programmer’s calculator. It only does integer math (no decimals), but it is designed so that it can always be called up within any program. It’s very small, and works with most graphic cards.

colacalc.zip 2/22/97

Band Names

Perhaps the most useless program ever written, it generates random names for rock bands.

bandname.zip 2/22/97
